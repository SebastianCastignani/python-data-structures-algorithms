# Estructuras de Datos y Algoritmos en Python

### Descripción
Este repositorio contiene una colección de ejercicios prácticos y modelos de parcial realizados en la materia de **Estructura de Datos** de la **Licenciatura en Informática (UNAHUR)**.


### Contenido 
* **01-fundamentos:** Introducción a Python y manejo de Arreglos.
* **02-estructuras-lineales:** Tipos de Datos Abstractos (TDA), Pilas, Colas y Listas Enlazadas.
* **03-recursividad-y-arboles:** Recursividad, Diccionarios, Conjuntos y Árboles Binarios.
* **04-examenes:** Modelos de parcial y ejercicios de examen.

### Stack Tecnológico
* **Lenguaje:** Python
* **Entorno:** Jupyter Notebooks / Google Colab
* **Librerías:** Numpy, Graphviz

### Cómo usar
Los archivos están disponibles en formato `.ipynb`. Pueden ejecutarse directamente en **Google Colab** o de forma local utilizando **Jupyter Lab** o la extensión de Notebooks en **VS Code**.
